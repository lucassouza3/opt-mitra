MITRARR Clean Rebuild
=====================

Repositório destinado à nova implementação do MITRARR seguindo o PRD em docs/PRD.md.

Estrutura inicial:
- apps/acquisition
- apps/normalization
- apps/distribution
- apps/monitoring
- core (entidades, casos de uso, interfaces)
- infra (adapters, scheduler, cli)

Use poetry install para preparar o ambiente quando o pyproject estiver configurado.
